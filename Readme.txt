"Pre-harvest maize fungal microbiome and mycotoxin contamination: Case of Zambia's Different rainfall patterns" (Published, 31st March 2023).

- The 'rds' files are saved objects to run in the RScript "Mycobiome.R". The files were generated using the pipeline divisive amplicon denoising algorithym (DADA) V2. Run code for DADA2 may be found at https://benjjneb.github.io/dada2/tutorial_1_8.html

Csv file "Mycobiome" is full fungal microbiome of maize samples including aflatoxin/fumonisin-B1.
